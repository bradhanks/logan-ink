# logan.ink Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build and deploy logan.ink — a portfolio/travelogue/grant-resource site for Logan Hanks that reads to a top grad-admissions committee as the work of someone in command of the population-science field, and ranks organically (SEO/AEO) for grant searches.

**Architecture:** Next.js 16 App Router with **Cache Components / PPR** (not classic `export const revalidate` ISR — see Caching constraint below) + React 19 + TypeScript + Tailwind v4, design tokens ported from the Claude Design export. Sanity (hosted) supplies content via a block-based page builder + Presentation visual editing. Pure-function modules carry the testable logic (Mancala engine, feed clients, schema/SEO helpers); UI is built on top and verified by build + manual check. Live "Field Feed" pulls keyless public APIs (PubMed, NIH RePORTER, ORCID, RSS, Bluesky) with long server-side caching. An MCP endpoint exposes the same data to agents. Deploy to Vercel, domain logan.ink.

**Tech Stack:** Next.js 16.2.9, React 19.2.4, TypeScript 5, Tailwind v4, next-sanity/sanity, `@next/third-parties` (GTM-TXP27537), `mcp-handler` + `@modelcontextprotocol/sdk`, Buttondown (newsletter), Resend (contact), Vitest (unit tests), Playwright (smoke, optional).

## Global Constraints

- Next.js **16.2.9**, React **19.2.7** (patch-bumped from 19.2.4 to satisfy the Sanity stack's `@portabletext/editor` peer; verified single-version resolution), TypeScript, Tailwind **v4**, Node 24+ (dev on 26). pnpm.
- Theme: resolve from `prefers-color-scheme` + manual toggle persisted to `localStorage`; inline no-flash script. Full `prefers-reduced-motion` support.
- **Caching = Cache Components (`cacheComponents: true`), NOT classic ISR.** The perf/infra foundation enables Partial Prerendering, so `export const revalidate`, `revalidatePath`, and bare `revalidateTag(tag)` are unavailable. Cache all server data with the shared toolkit in `lib/cache.ts`: `"use cache"` + `cacheLife(CACHE.<profile>)` + `cacheTag(cacheTags.<x>)`; invalidate via `invalidate.<x>()` (background) or `{ immediate: true }` (same-request). This rewrites the data-layer mechanics in Tasks 1.2 (`sanityFetch` → wrap in `"use cache"`), 1.4 (revalidate webhook → `invalidate.page(slug)`), 1.3/2.x (page fetches → `cacheLife`/`cacheTag` instead of `export const revalidate`), and 4.2 (Field Feed → `cacheLife(CACHE.feed)`). Anything reading `cookies()`/`headers()`/`searchParams` must sit in `<Suspense>` and stay out of `"use cache"` functions. See `docs/decisions/0001-caching-cache-components.md`.
- **No fabricated data on production.** Mock fixtures only in dev; prod shows real cached data + "updated" timestamp, or honest empty/error states. Never fake feed entries.
- **No gamified author-credit counter** anywhere public. Ambition is narrative only. The literal "10 credits" goal lives only in `STRATEGY-target-professors.md` (private, never shipped/indexed).
- **Public data only in Sanity/MCP.** The MCP endpoint is public and unauthenticated; therefore Sanity contains *only* publishable content. The professor-target strategy, diversity-as-leverage notes, and any private analysis NEVER enter Sanity, the repo (beyond the gitignored strategy file), or any MCP-reachable surface.
- **Structured data only for real, verifiable items.** No `ScholarlyArticle` JSON-LD for placeholder pubs.
- **Index hygiene:** `/studio`, `/api/*`, `/mcp/*`, draft routes, and all preview/branch deployments are `noindex`/robots-excluded. Only production canonical URLs are indexable.
- **Inclusiveness stays subtle** — color/typographic warmth + sparing trans-flag accent; never a banner.
- **Interaction restraint:** native cursor (no follower ring); micro-interactions hint at function, not cleverness; no profile-image hover gimmick.
- Real-people photos (heroes list) must not use unlicensed headshots — link-out / initials / illustration only.
- GTM container is **GTM-TXP27537** (auto-injected via `@next/third-parties`, head + noscript).
- Every invented/assumed fact about Logan is recorded in `CONFIRM-THESE-FACTS.md` at repo root.
- Spec of record: `docs/superpowers/specs/2026-06-17-logan-ink-portfolio-design.md`.
